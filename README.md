# 概要
7セグLEDとラッチICで構成した基板とRaspberryPiを接続して作成したシンプルな時計です。  
RaspberryPi内の内臓時計の時刻を表示しており、WiFi経由でNTPサーバに同期しているため正確な時刻が確認できます。
  
time.cpp が、RaspberryPi上で動作させるソースコード(C++言語)です。
